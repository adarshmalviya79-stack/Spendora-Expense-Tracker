# 💰 Spendora – Smart Expense Tracker

Spendora is a modern and intelligent expense tracking platform designed to help users take full control of their finances. With a clean UI and powerful features, it simplifies money management and transforms raw financial data into meaningful insights.

---

## 🚀 Key Features

### 📊 Real-Time Financial Dashboard
- Get an instant overview of your income, expenses, and remaining balance  
- Clean and interactive UI  
- Visual graphs and cards for better clarity  

### 💸 Expense & Income Management
- Add, edit, and delete transactions  
- Categorize expenses (Food, Rent, etc.)  
- Maintain complete transaction history  

### 📈 Smart Budgeting System
- Set monthly budgets  
- Track remaining budget in real-time  
- Get alerts when budget is exceeded  

### 🤖 AI-Powered Insights
- Analyze spending behavior automatically  
- Suggest ways to reduce expenses  
- Identify high-spending categories  

### 👥 Group Expense Management
- Create or join groups via invite code  
- Split and manage shared expenses  
- Track contributions easily  

### 🔐 Secure & Reliable
- Login/Signup authentication  
- REST API based backend  
- Scalable and smooth performance  

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, ReactJS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **API:** RESTful APIs
